# Simple Style Guide

A minimal style guide for teaching purposes.

## Colors

- **Primary**: `#3498db` (Blue)
- **Success**: `#27ae60` (Green)
- **Danger**: `#e74c3c` (Red)
- **Text**: `#2c3e50` (Dark Gray)
- **Background**: `#f5f5f5` (Light Gray)

## Typography

- **Font**: System fonts (Arial, Helvetica, sans-serif)
- **Sizes**: Small `14px`, Normal `16px`, Large `24px`
- **Weights**: Normal `400`, Bold `700`

## Spacing

Use multiples of 8px: `8px`, `16px`, `24px`, `32px`

## Components

**Buttons**: 
- Padding: `12px 24px`
- Border radius: `4px`
- Blue background for primary actions

**Cards**:
- White background
- Border radius: `8px`
- Shadow: `0 2px 8px rgba(0,0,0,0.1)`

## Rules

1. Use colors from the palette above
2. Keep spacing consistent (8px increments)
3. Make buttons and links easy to click
4. Test on mobile devices
